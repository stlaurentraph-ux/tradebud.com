# Demo Readiness Checklist

- [ ] Producer inbox has at least 2 actionable items.
- [ ] Supplier can complete `DRAFT -> READY -> SEALED -> SUBMITTED`.
- [ ] Buyer can `ACCEPT` and `REJECT` a submitted package.
- [ ] Sponsor dashboard shows governance and escalation context.
- [ ] Admin can invite user and update role.
- [ ] Error states show retry CTA on key pages.
- [ ] Demo runbook rehearsed end-to-end in under 10 minutes.
