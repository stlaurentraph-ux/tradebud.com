## Start with identity preservation

EUDR prohibits mass-balance chains of custody that obscure origin coordinates. Before you scale batch volume, confirm your data model tracks **identity-preserved** lots from farm to shipment.

## Eight steps exporters can take now

- Inventory which farmers and plots feed each export lane
- Standardize GeoID capture (polygon or point per hectare rules)
- Collect timestamped, geo-tagged field photos for ground-truth evidence
- Run yield-cap checks against verified plot capacity
- Segregate flagged plots before they enter a shipment pool
- Run a zero-risk pre-flight check across the full batch pool
- Prepare DDS artifacts from structured source data — not manual re-entry
- Plan TRACES NT submission as SOAP/XML middleware, not a simple JSON POST
