# P0-03 Pilot Kickoff Checklist

## Purpose

Start the required 3-month MVP pilot in a controlled, auditable way before Release 2.

## Pilot Success Targets (from execution board)

- One Tier 2 cooperative onboarded.
- At least 20 farmers active in pilot scope.
- At least 1 completed batch through target flow.
- One Tier 3 buyer involved in downstream workflow.

## Pre-Kickoff Checklist

- [ ] Pilot cooperative identified and confirmed in writing.
- [ ] Tier 3 buyer confirmed in writing.
- [ ] Farmer cohort target list (`>=20`) prepared.
- [ ] Baseline workflow map reviewed (field capture -> batch -> buyer handoff).
- [ ] Pilot start date and end date (3-month window) set.
- [ ] Owners assigned:
  - Pilot owner:
  - Operations owner:
  - Data/analytics owner:

## Scope and Guardrails

- [ ] Pilot scope boundaries documented (regions, products, languages, channels).
- [ ] Out-of-scope requests list documented to avoid uncontrolled expansion.
- [ ] Exception escalation path defined.

## Evidence and Reporting Setup

- [ ] Evidence log initialized in `product-os/04-quality/p0-03-pilot-evidence-log.md`.
- [ ] Weekly checkpoint cadence scheduled.
- [ ] End-of-pilot report owner assigned.
